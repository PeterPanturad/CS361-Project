Instructions on how to run code for password.py

In the .zip file, there will be a file included called 'processed_passwords.txt.' This is our dataset.
The dataset contains 900,000 lines from the haveibeenpwned dataset. It has been processed already to remove the '0x' and ':00'


1. Install python library: numpy. If on Linux, use "sudo apt install python3-numpy," otherwise "pip isntall numpy"
2. Run in terminal: python3 password.py
3. View output!

Output:
Total items to be inserted (in our dataset it is 450000)
False positive rate (in decimal)
Number of bits in the array
Number of hash functions

Queries training set:
Amount of hits for each password hashes | Avg runtime per 1000 queries (ms)


Queries non training set:
Amount of hits for each password hashes | Avg runtime per 1000 queries (ms)
