Instructions on how to run code for threewaymerger.py


Numpy
1. Install python libraries: numpy and matplotlib via pip install "pip install matplotlib" and "pip install numpy"
2. Run in jupyter-notebooks or Colab
3. View output in console and at the end see graphs once both integer types are done.

The array that was sorted
